# 基于 RDK X5 的陆空协同智慧农业巡检与精准作业系统

## 项目简介

本项目面向温室大棚、果园和智慧农场等场景，基于 RDK X5 构建陆空协同智慧农业巡检与精准作业系统。系统由无人机巡检端、地面小车作业端和 RDK 地面站组成，可实现空中巡检、作物异常识别、地面近距离复查、精准浇水施药、异常记录和可视化管理等功能。

项目采用“空中全局巡查 + 地面近距离复查 + 精准水肥药作业 + 地面站可视化管理”的总体思路。无人机负责大范围巡检和异常初筛，地面小车负责近距离复查与定点作业，地面站负责实时显示、任务管理和数据留存，从而提升农业巡检效率、作业精度和智能化管理水平。

本项目适用于以下场景：

- 温室大棚作物巡检

- 智慧农场自动化管理

- 精准浇水、施肥和喷药作业
- 农业机器人教学与竞赛展示

## 系统组成

系统主要由三部分组成：

### 1. 无人机巡检端

无人机负责空中大范围巡检，采集作物图像，并对疑似异常区域进行初步识别。发现异常后，将图像、异常类别、置信度和位置信息上传至地面站。

### 2. 地面小车作业端

地面小车以 RDK X5 作为上位机核心，负责图像采集、目标识别、任务调度、激光雷达/IMU 数据处理和通信转发。STM32F407VET6 作为下位机，负责底盘运动、舵机云台、双水泵和急停保护等实时控制。

### 3. RDK 地面站

地面站用于显示无人机和小车画面，接收异常报告，展示异常类别、置信度、来源、位置和图片信息，并保存历史记录，实现农业巡检和作业过程的数据留痕。

## 主要功能

### 1. 空中全局巡检

无人机对温室或果园区域进行快速巡查，发现疑似异常作物区域，提高巡检覆盖范围和效率。

### 2. 地面近距离复查

地面小车接收异常任务后，前往目标区域进行近距离图像采集和复查，提高异常判断准确性。

### 3. 作物异常识别

系统可对黄叶、枯叶、缺水、病害等异常现象进行识别，并输出异常类别、置信度和截图信息。

### 4. 精准浇水施药

地面小车搭载双水泵系统，清水泵和药液/营养液泵分开控制，可根据异常类型执行短时脉冲喷洒，实现精准作业。

### 5. 地面站可视化管理

地面站可实时显示巡检画面、异常信息、任务状态和历史记录，方便操作人员统一管理。

### 6. 安全保护机制

系统设计了急停、通信超时自动停车、水泵默认关闭、单泵关闭和动作限时等保护机制，减少误动作风险。

## 硬件组成

| 模块 | 硬件 | 功能 |
|---|---|---|
| 小车上位机 | RDK X5 | 图像处理、ROS2、任务调度、通信转发 |
| 小车下位机 | STM32F407VET6 | 电机、舵机、水泵、急停控制 |
| 感知模块 | USB 摄像头、激光雷达、IMU | 图像采集、环境感知、姿态检测 |
| 执行模块 | 四路电机驱动、双水泵、继电器、舵机云台 | 底盘运动和精准喷洒 |
| 地面站 | RDK/显示屏/网络模块 | 视频显示、异常记录、任务管理 |
| 无人机端 | 飞控、Jetson NX、D435i/相机 | 空中巡检和异常初筛 |

## 系统工作流程

1. 无人机执行空中巡检任务；
2. 摄像头采集作物图像；
3. 系统识别疑似黄叶、枯叶、缺水或病害区域；
4. 异常信息上传至 RDK 地面站；
5. 地面小车接收异常任务；
6. 小车到达目标区域后进行近距离复查；
7. RDK X5 根据识别结果生成作业指令；
8. STM32F407VET6 控制底盘、云台和水泵执行动作；
9. 小车完成精准浇水、施肥或喷药；
10. 地面站保存异常报告和作业记录。

## RDK X5 在本项目中的作用

RDK X5 是本项目地面小车的智能中枢，主要承担以下任务：

- 运行 Ubuntu 22.04 和 ROS2 Humble；
- 接入摄像头、激光雷达和 IMU；
- 进行作物图像采集和异常识别；
- 接收地面站或无人机端上传的异常任务；
- 通过串口向 STM32F407VET6 发送控制指令；
- 管理小车运动、复查、喷洒和状态回传流程；
- 支持后续部署轻量化 AI 大模型，实现更智能的任务理解和作业决策。

## 项目创新点

1. 提出“空中全局巡检 + 地面近距离复查 + 定点精准作业”的陆空协同农业机器人方案；
2. 基于 RDK X5 构建端侧智能感知与任务调度系统；
3. 采用 RDK X5 + STM32F407VET6 的上下位机分层控制架构；
4. 设计双水泵比例喷洒方案，实现清水和药液/营养液分路控制；
5. 建立地面站异常报告系统，实现作业过程可视化和数据留痕；
6. 系统具备良好的模块化结构，便于后续扩展自动导航、AI 大模型和多机器人协同。

## 未来展望

未来可在小车端的 RDK X5 上部署轻量化 AI 大模型或多模态模型，使小车具备更强的自然语言理解、异常原因分析、智能作业推荐和报告生成能力。操作人员可通过语音或文字下达任务，系统结合摄像头、激光雷达、IMU、环境数据和历史记录，自动判断作物异常原因并推荐浇水、施肥或喷药策略，让小车从“自动执行”升级为“智能决策”。
